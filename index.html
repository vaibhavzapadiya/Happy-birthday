Use Case: Build a customer feedback form where the FeedbackService saves and fetches customer feedback. Business Logic: Upon form submission, save the feedback using the FeedbackService. The service should save feedback data and also fetch all submitted feedback to display in a table. UI: A form for customer name, feedback, and rating (1-5). Display all feedback in a table with the ability to edit or delete. Control IDs: Feedback Form: Customer Name: customerName Feedback Message: feedbackMessage Rating: feedbackRating Submit Button: submitFeedbackButton Feedback Table: Customer Name: feedbackCustomerNameColumn Feedback Message: feedbackMessageColumn Rating: feedbackRatingColumn Edit Button: editFeedbackButton Delete Button: deleteFeedbackButton
